# Supermarket Sales Dashboard

## Descripción del Proyecto

Este proyecto presenta un dashboard interactivo desarrollado en Tableau Public para analizar el desempeño de ventas de un supermercado.

El objetivo es transformar datos transaccionales en información útil para la toma de decisiones mediante indicadores clave y visualizaciones intuitivas.

---

## Objetivos del análisis

- Analizar las ventas totales generadas por el supermercado.
- Identificar el ticket promedio de compra.
- Medir el volumen total de transacciones.
- Evaluar las unidades vendidas.
- Comparar el desempeño entre ciudades.
- Analizar las categorías de productos más rentables.
- Comparar el comportamiento de diferentes tipos de clientes.
- Identificar diferencias de ventas por género.
- Comparar el desempeño de las sucursales.
